# X平台数据追踪器

一个Chrome浏览器扩展，用于追踪和统计您在X平台(x.com)上的活动数据。

## 功能特性

- ✅ 追踪点赞、转发活动
- ✅ 实时显示今日统计数据
- ✅ 显示最近活动列表
- ✅ 本地数据存储
- ✅ 数据清空功能

## 安装方法

1. 下载或克隆此项目
2. 打开Chrome浏览器，访问 `chrome://extensions/`
3. 启用"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹

## 使用说明

1. 安装扩展后，访问 [x.com](https://x.com)
2. 正常使用X平台进行点赞、转发等操作
3. 点击浏览器工具栏中的扩展图标查看统计数据
